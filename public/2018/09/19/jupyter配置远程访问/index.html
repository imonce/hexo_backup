安装jupyter

pip install ipython
pip install jupyter

生成jupyter配置文件

jupyter notebook –generate-config

自动生成密码

jupyter notebook password

获取密码

cat /home/xm/.jupyter/jupyter_notebook_config.json

修改配置文件

vim /home/xm/.jupyter/jupyter_notebook_config.py
